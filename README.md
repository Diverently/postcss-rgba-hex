# postcss-rgba-hex

> [PostCSS](https://github.com/postcss/postcss) plugin that replaces RGB(a) values with hexadecimal analogues


## Processing example

this code

```
.element {
    border: 2px solid rgba(0,0,0,.2);
}
```

converts to

```
.element {
    border: 2px solid #ccc;
}
```
